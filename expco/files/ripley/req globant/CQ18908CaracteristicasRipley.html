CQ18908 - Características Ripley

Justificación del Requerimiento

(Lista de objetivos a los cuales aporta)

  1. Definir la creación de una nueva característica personalizada para el portafolio de características de Ripley
  2. Crear una característica que proporcione una definición de si  un individuo tiene o no Tarjetas de Crédito Abiertas con Ripley

(Relación con otros requerimientos)

  1. Relación con los documentos proporcionados por Experian:
  1. Características Ripley CQ18908.docx
  2. Sol Caract RIPLEY 03Mar2014.xlsx

Descripción de las Necesidades Funcionales

  1. Característica CO01NUM996RO029 - Tarjetas de Crédito Abiertas

Se requiere la creación de una nueva característica personalizada para la entidad Ripley, que proporcione la información que permita identificar si un individuo tiene o no Tarjetas de Crédito Abiertas con la entidad.

Formato

Numérico

Longitud

3, 3-Entero, 0-Decimal

Portafolio

RO_01

Tipo de Característica

Personalizada

Precondiciones

N/A

Pseudocódigo

FOR EACH Individuo:

Set CO01NUM996RO029 to zero

Set VALUE to zero

FOR EACH Cuenta en buró:

IF Cuenta pertenece a Portafolio

IF Código suscriptor in (‘Código suscriptor Ripley)

IF Cuenta is Abierta

Set VALUE to one

END IF

END IF

END IF

END FOR Cuenta

Set CO01NUM996RO029 to VALUE **Valor Final Característica**

END FOR Individuo

Esta variable hace referencia a la búsqueda en el maestro de crédito de cada cliente, por Tarjetas de Crédito abiertas con Ripley. Nit = 900517323 código suscriptor= 020030 

Sin embargo, esta característica debe quedar como un indicador codificado, donde sólo a través de una interpretación específica se pueda obtener esta información.

1-Tiene Tarjeta de Crédito con Ripley

0-No tiene Tarjeta de Crédito con Ripley

Anexos

(Información general que aplica a la descripción del requerimiento)

Conceptos Involucrados

A

Abierta

Una Cuenta abierta es:

Si Tipo de Cuenta in ( ‘01’, ‘51’)

1. Aquella cuyo Estado posee alguno de los siguientes códigos ‘01’, ‘06’, ‘07’, ‘09’.

Si Tipo de Cuenta not in ( ‘01’, ‘51’)

1. Aquella cuyo Estado posee alguno de los siguientes códigos ‘01’, ‘13’ al ‘41’, ‘45’, ‘47’.

C

Código Suscriptor

Campo COD_SUSCRIPTOR de los Maestros de Crédito

Código suscriptor Ripley

Son todos los códigos de suscriptor asociados a la Entidad Ripley identificado con Nit. 900517323 Los cuales son : 020030.

Estos códigos de suscriptor pueden modificarse a criterio de Datacrédito o la propia entidad y debe garantizarse que el desarrollo no se vea afectado con dichos cambios

Cuenta

Entrada o registro en el Hábito de Pago que define la historia de la deuda que una persona posee con una entidad crediticia siempre y cuando ésta no se encuentre Bloqueada. También se Conoce como Obligación.

I

Individuo

Persona cuyo expediente de crédito es sujeto de estudio.

P

Portafolio

La asignación de Portafolio debe realizarse de forma secuencial, de la siguiente manera:

If Portafolio=’CO’

  Else if Portafolio=’RO’

    Else if Portafolio=’VE’

      Else if Portafolio=’HP’

        Else if Portafolio=’IN’

          Else if Portafolio=’CC’

            Else if Portafolio=’AH’

              Else if Portafolio=’CT’

Else if Portafolio=’OT’

Cumpliendo cada una de las especificaciones de cada portafolio.

Dada la especificación dada por Helm Bank, es posible que algunas cuentas queden por fuera de los portafolios mencionados.

Criterios de Aceptación

(Lista de criterios o condiciones que son la base para la aceptación del requerimiento).

  1. Para individuos que poseen TDC abiertas con Ripley el resultado de la característica es 1
  2. Para individuos que NO poseen TDC abiertas con Ripley el resultado de la característica es 0
  3. Los datos para la certificación de pruebas son suministrados por el área de Decision Analytics
