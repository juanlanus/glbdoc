Definición

El programa de prueba que nos están pidiendo tiene las siguientes características:

- Tiene un archivo secuencial de entrada ordenado de forma aleatoria con los siguientes datos:

    TIP-ID      PIC 9(01).
    NUM-ID      PIC 9(11).

- Con cada registro de este archivo se debe ir a buscar en el archivo SCODAPJA y generar un registro de salida con los datos encontrados.

Te envío adjunto el archivo SCODAPJA, el archivo .FD y .FS y un programa que se puede tomar como base para hacer el desarrollo.
